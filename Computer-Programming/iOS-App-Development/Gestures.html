UIGestureRecognisers is the abstract class, therefore it can be used directly. The Gesture that the Apple provides are
  • UIPanGestures.
  • UISwipeGestures.
  • UIPinchGestures.
  • UITapGestures.

Talking about the gesture we will first talk about thePan gesture:

Pan Gesture

Here first we will talk about the gesture that is pan gesture, the pan or in other words drag. This gesture is handy when you want to allow your users to drag views around the screen. In this part, except for implementing the necessary code that will enable our app to support the pan gesture, we will meet a special method of the UIPanGestureRecognizer class, named velocityInView. This method returns a CGPoint value, and informs us about how many points per second the dragged view travels in both the horizontal and vertical axis while dragging. Let’s Begin the Coding Part For using the pan gesture we will build a view and will paste a Label on that view.After pasting the label the view will appear like this.

Now moving onto the coding part We need to go the Viewcontroller.m file and write the following code in the program below the ViewController.m file and under the ViewDidLoad we need to write the following code. 
                self.view.backgroundColor = [UIColor whiteColor];
                /* Let's first create a label */
                CGRect labelFrame = CGRectMake(0.0f, /* X */

                self.panGestureRecognizer.maximumNumberOfTouches = 1;
                /* Add it to the view */
                [self.helloWorldLabel addGestureRecognizer:self.panGestureRecognizer];

Then I have set the colour of text in label as white colour. And kept the alignment as centre. Here alignment means where we want to place a particular thing. The alignment could be in centre, left or right Applying the Pinch Gesture..... After this we need to add the pan gesture. Here I have added the Pan gesture with the maximum and minimum number of touch required for pan gesture. 
                 - (void) handlePanGestures:(UIPanGestureRecognizer*)paramSender
                  {

Pinch Gesture


Taking about the gestures here we have another gesture known as pinch gesture. Pinch gesture is used to actually change the size of the view. The pinch gesture is useful in changing the transformation of the screen by scaling it upward or downwards.one of the example of the pinch gesture is in your google app where we pinch in and pinch out. Here we do not need to add any image view we will work by using a view. For using the pinch gesture in the application we need to use two fingers.

Moving onto the coding part Here also we have taken the view and worked on that. Here I have the class with the name ViewController. Beginning the important part we have a view.

First we need to go the ViewController.h file and write the Following code

For that we have to include the following code in our program. 
                - (void) handlePinches:(UIPinchGestureRecognizer*)paramSender{
               if (paramSender.state == UIGestureRecognizerStateEnded){ self.currentScale = paramSender.scale;
               } else if (paramSender.state == UIGestureRecognizerStateBegan && self.currentScale != 0.0f){
               paramSender.scale = self.currentScale; }
               if (paramSender.scale != NAN && paramSender.scale != 0.0){
               paramSender.view.transform = CGAffineTransformMakeScale(paramSender.scale,paramSender.scale);
               } }

Swipe Gesture

After the two Gestures I will tell you one of the other gesture that we use in iOS application that is Swipe gesture. One of the most important thing that the iOS application uses is swipe gesture. Swiping can be done in any direction. A swipe gesture object can trigger and action in one direction only. Many of the application today use swipe gesture. The swipe gesture improves the user interface of the application. Let’s begin with the coding part Now let’s start the Coding part.
