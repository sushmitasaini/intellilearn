iOS App Development Tutorials:PageController in iOS App Development

Posted By:Gaurav Saini

Sometimes when we have multiple images and you want to add them in your application we use the concept of page controller. The page controller help us to slide multiple image on a view. The page controller looks like three dot at the bottom of your image. Now moving on the coding part of the application we have to include the following part
  • 1-UISliderView
  • 2-PageController


Now let’s begin the coding part. Here I have created the cocoa touch class with the name View Controller. Moving on to the next part we will create the property of the UISliderView along with PageController in the ViewController. 
                @interface ViewController : UIViewController
                @property (weak, nonatomic) IBOutlet UIPageControl *pageControll;
                @property (weak, nonatomic) IBOutlet UIScrollView *scrollView;
