One of the important feature that we use today is map in our day to day life. So it become important for a developer to integrate map in his application. Therefore x code provides a framework which can be used for integrating the map in iOS. MapKit is a neat API, comes with the iOS SDK, that allows you to display maps, navigate through maps, add annotations for specific locations, add overlays on existing maps, etc. In this tutorial, we’ll walk you though the basics of the API and show you how to place a pin on the map.
