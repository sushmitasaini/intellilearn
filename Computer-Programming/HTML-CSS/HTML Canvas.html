HTML/CSS Development Tutorials:HTML Canvas

Posted By:Ankita

  • Share on Facebook
  • Tweet
  • Share on Google+
The HTML <canvas> element is specifically used to draw graphics on the web page. It is defined in HTMLcode using width and height attributes. A canvas is a rectangular area on an HTML page in which it has no border and no content.The proper usability of the canvas element is accomplished by taking advantage of the HTML5 Canvas API which is used by writing JavaScript that can access the canvas area through a full set of drawing functionsand however allowing for dynamically generated graphics.
IMPORTANT: The canvas context is an object with properties and methodologies by which user can use torender graphics inside the canvas element. The context can be 2d or webgl (3d).
We can get the Canvas element in DOM by using getElementById() method as follows:
var canvas = document.getElementById("mycanvas");

                    Lets implement a small example to understand this better
                    <!DOCTYPE HTML>

                    <html>

                    <head>

                    <style>

                    #mycanvas{border:2px solid blue;}

                    </style>

                    </head>

                    <body>

                    <canvas id="mycanvas" width="100" height="100">

                    </canvas>

                    </body>

                    </html>

Output

Lets take an Example

                    Lets try this one
                    <!DOCTYPE html>

                    <html>

                    <body>

                    <canvas id="myCanvas" width="200" height="200">

                    </canvas>

                    <script>

                    var c = document.getElementById("myCanvas");

                    var context = c.getContext("2d");

                    context.fillStyle = "#FF0000";

                    context.fillRect(10,20,100,75);

                    </script>

                    </body>

                    </html>

OUTPUT

In order to draw a line, we use four canvas API methods.
* We start with the beginPath() method which instructs the browser to prepare itself to draw a new path.
* Then, we use the moveTo(x, y) method to set the line’s starting point.
* Followed by lineTo(x, y) method which sets the ending point and draws the line by connecting the two points.
At this point the line will be drawn, but it’s still invisible. To make it visible we use the stroke() method to display the line with the default black color.
Lets try this one to understand this better 
                    <!DOCTYPE html>

                    <html>

                    <body>

                    <canvas id="myCanvas" width="200" height="200">

                    </canvas>

                    <script>

                    var c = document.getElementById("myCanvas");

                    var context = c.getContext("2d");

                    context.beginPath();

                    context.moveTo(50, 50);

                    context.lineTo(250, 150);

                    context.stroke();

                    </script>

                    </body>

                    </html>
