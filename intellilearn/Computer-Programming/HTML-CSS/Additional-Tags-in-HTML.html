Additional Tags in HTML

<article>Tag
It is the independent tag which is self-contained. They are used while writing: Blogs, Comments or any News update. 
                <!DOCTYPE html>
                <html>
                <body>
                <article>
                <h1>Intellilearn</h1>
                <p>IntelliLearn is a Blog site where blogger can Learn & Earn by sharing blog posts on different domains like iOS Development, Windows App Development, HTML and many more.....</p>
                </article>
                </body>
                </html>
OUTPUT



<figcaption>Tag

It is used to define the caption for the figure.It can used to give description about the figure used. 
                <!DOCTYPE html>
                <html>
                <body>
                <figure>
                <img src="garden_rose.jpg" alt="Red Rose" width="304" height="228">
                <figcaption>Fig.1 - A view of the rose garden</figcaption >
                </figure>
                </body>
                </html>

OUTPUT



<section>Tag

This tag is used to divide the document into different sections. Like into header, footer, different chapters etc. 
               <!DOCTYPE html>
                <html>
                <body>
               <section>
                <h1>Cannes</h1>
                <p> Famous city at the French Riviera, the Côte d'Azur in France, tourist attraction and host of the annual Cannes Film Festival. Cannes has a population of about 68,000 inhabitants, official language is French. 

               </section>
               <section>
               <h1>Chicago</h1>
               <p>Chicago was founded in 1833, today the city is a major transportation and telecommunications hub and a center for business and finance, it is listed as one of the world's top ten Global Financial Centers. It is rated as an alpha+ global city that means it is considered to be an important node in the global economic system. 

               </section>
               </body>
               </html>




<nav>Tag
This tag is used to set the navigation links. 
                <!DOCTYPE html>
                <html>
                <body>
                <nav>
                <a href="http://intellilearn.in/Blogs/ios/IosPage.aspx">iOS Development</a> |
                <a href="http://intellilearn.in/Blogs/angular/AngularPage.aspx">Angular JS</a> |

                <a href="http://intellilearn.in/Blogs/python/PythonPage.aspx"> Python</a>
               </nav>
               </body>
               </html>
