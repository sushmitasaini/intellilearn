HTML5/CSS3 Development Tutorial:How To Use CSS3

Posted By:Gaurav Saini

  • Share on Facebook
  • Tweet
  • Share on Google+
There are three ways of inserting a style sheet
  • External style sheet
  • Internal style sheet
  • Inline style

External Style Sheet 
               
                <head>
                <link rel="stylesheet" type="text/css" href="mystyle.css">
                </head>

An external style sheet can be written in any text editor. The file should not contain any html tags. The style sheet file must be saved with a .css extension. 
              body {
              background-color: lightblue;
                   }
              h1 {
              color: navy;
              margin-left: 20px;
               }

This part of code is written in the new file with extension.css

Internal Style Sheet

                <!DOCTYPE html>
                <html>
                <head>
                <style>
                    body {
                        background-color: Green;
                    }
                    h1 {
                        color: Blue;
                    }
                </style>
                </head>
                <body>
                <h1>This is a heading of a paragraph</h1>
                <p>This is a paragraph one.</p>
                </body>
                </html>

Inline Styles


An inline style may be used to apply a unique style for a single element. This is a very old method used in CSS 3. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property. 
                 <!DOCTYPE html >
                 <html >
                 <body >
                 <h1 style="color:Green;margin-left:10px;"> This is a heading 1.</h1>
                 <h1 style=”color:yellow”>This is heading 2</h2>
                 <p>This is a paragraph.</p>
                    </body>
                    </html>
