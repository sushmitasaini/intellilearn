iOS App Development Tutorials:Localytics in iOS App Development

Posted By:Gaurav Saini

  • Share on Facebook
  • Tweet
  • Share on Google+
Localytics was conceived in 2009 in Boston, Massachusetts with one simple goal in mind: to help our customers build stronger relationships with their mobile and web app users through our analytics and marketing platform.
It's a win-win for brands and end-users alike. App publishers create better app experiences, leading to passionate users that ultimately drive engagement, loyalty and revenue.
Many of the world's best-known brands including eBay, Salesforce.com, Microsoft, and The New York Times have benefited from using our solutions to create insight into their app users and leverage that knowledge to predict and create more personalized user engagement.
Localytics currently supports more than 6,000 customers and reaches 37,000 apps, 2.7 billion devices and 120 billion data points monthly

How To Use Localytics in iOS


Step For Applying Localytics

First of all create a Account onLocalytics

Step For Applying Localytics

Choose Setting Button on Localytics Websites.

Step 4 For Applying Localytics


After that Click on Create a App



Step 5 For Applying Localytics


Open Xcode, select New > Project... from the File menu, and choose the Single View Application template from the iOS > Application category.


Step 6 For Applying Localytics

Step to install pod file
Open Terminal and write a command… $ sudo gem install -n /usr/local/bin cocoapods or $ sudo gem install cocoapods
After that select the path of your project .. and write command.
pod init.

Now a Pod file will create..
and open that file and add a Line pod 'Localytics', '~> 3.0’


and now open and work on xcworkspace file.

Step 7 for Applying Localytics


Paste into your application's delegate file, the file whose default name is ‘AppDelegate.m’ #import "Localytics.h" Add the following line to the start of didFinishLaunchingWithOptions. This integrates the Localytics library into the app, opens the session, and causes an upload on app start 

               - (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions
                 {
               [Localytics autoIntegrate:@"61812930e1ab675c6ac349a-f5091e80-9025-11e5-0b39-008b20abc1fa" launchOptions:launchOptions];
               return YES;
                  }
                //  [Localytics autoIntegrate:@"61812930e1ab675c6ac349a-f5091e80-9025-11e5-0b39-008b20abc1fa" launchOptions:launchOptions];


This code will be different for different app and can be found on localytics.Localytics integration is done. Run your App and you check it on Dashboard
