What is Webservices

Web services are the important part of the iOS. Web services are used when we give some request and a response is generated from the server. A web service is any piece of information that makes itself available over internet. A web service is a collection of open protocol and standards used for exchanging the data between application or software. There are basically two method of sending request • Rest • Soap In rest we use Json and in soap we use xml. Before staring the coding portion it is necessary to make a profile in the Github.
