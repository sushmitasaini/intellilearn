Python Tutorial:How To Work With String in Python

Posted By:Garvit Rajput

  • Share on Facebook
  • Tweet
  • Share on Google+
Sometime a problem occurs with us that is we want to concatenate string and number it throws an error saying that can no concatenate string and number. To overcome that problem we need to store number into string variable and doing so is knows as typecasting. Typecasting is also known as Boxing-Unboxing.
Let’s see a real life problem. If we want to take user’s age and then we want to pring it with greetings. 

                    age = input("Enter Your Age :   ");
                    
                    Enter Your Age :   18
                    
                    print "Hello user, You are "+age
                    
                    Traceback (most recent call last):
                    
                    File "<pyshell#19>
                        
                        ", line 1, in <module>
                                                        
                            print "Hello user, You are "+age


                            TypeError: cannot concatenate 'str' and 'int' objects
Here it thrown error of typecasting.
Now let’s typecast the above variable to string. 

                    age=str(age)
                    
                    print "Hello User, You are  "+age
                    
                    Hello User, You are  18
The typecasting is done by specifying the datatype in which we want to convert and passing it data variable.
Typecasting to float: 

                    age = float(age)
Tyecasting to int:

age = int(age)
